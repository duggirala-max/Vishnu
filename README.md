# Noor Al Reef "Silk Road" Project

// Built by Duggirala for Vishnu Vardhan

## Overview
Noor Al Reef General Trading LLC's digital presence is a premium B2B portal designed to bridge the trade gap between India and Dubai. This project utilizes a modern tech stack to deliver a high-performance, visually stunning experience that reflects the company's commitment to quality and logistical excellence.

## Tech Stack
- **Framework:** React 19 (TypeScript)
- **Bundler:** Vite
- **Styling:** Tailwind CSS (Vanilla CSS for base overrides)
- **Animations:** Framer Motion for UI micro-animations
- **3D Visualization:** Cobe for the interactive global shipping map
- **Typography:** Poppins Light (Body) & Cera Pro Medium (Headers)

## Brand Identity
- **Primary Canvas:** Pure White (#ffffff)
- **Primary Emerald:** #00685f (Used for headings, icons, and accents)
- **Accent Orange:** #e17726 (Used for primary Calls to Action and highlights)

## Getting Started
1. Navigate to the `noor-al-reef` directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Run `npm run build` for a production-ready bundle.
